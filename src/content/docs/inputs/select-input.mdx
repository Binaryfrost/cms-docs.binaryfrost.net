## Settings

### Options

The options to be shown in the Input. Each option has two required inputs: value and label.

### Required

Whether this Input needs to have a value. Required Inputs without a value will prevent the Collection Entry from being saved.

<InputExample image="select-settings" isSetting />

## API Response

The value will be returned as a string.
