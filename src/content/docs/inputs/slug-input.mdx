The Slug Input generates a slug based on another field.

<InputExample image="slug" />

:::tip
Before publishing a draft, update the slug by clicking the refresh button to the right of the input field. This ensures that the date placeholders, if in use, reflect the publishing date.
:::

## Settings

### Field Name

The field name of the Collection Input the slug will be based on.

### Max Length

The maximum length of the slug.

### Required

If true, this Input needs to have a value.

### Before

If set, this value will be prepended to the slug with the placeholder values replaced.

<InputExample image="slug-settings" isSetting />

## API Response

The value will be returned as a string.
